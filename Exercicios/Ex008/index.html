Texto Marcado

agora vamos marcar uma palavra como se fossemos usar marca-texto, basta usar o comando <mark>

Texto Grande e Pequeno

Estramos criando um texto grande e um texto pequeno neste parágrafo usando as tags <big> (obsoleta) e <small> (não obsoleta)

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Ps: Utilizamos a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Ps: Utilizando a tag <ins> (semântica)

Existe também o sublinhado com a tag <u> (não semântica)

Texto Subscrito
